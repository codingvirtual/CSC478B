1 ⟡ OVERVIEW

Mirror is a backup utility designed to help you keep a personalized set of files and/or folders mirrored to a secondary location. Our software was developed to make the backup procedure as simple and as straightforward as possible. It gives individuals, like you and I, a free alternative to the commercialized systems made for corporations.

With Mirror, backing up your computer becomes an enjoyable task instead of a tedious one. You'll have access to all the essentials (cloud-based storage, backup scheduling, etc.) without all the excess fluff that slows you down and makes you less productive.

Now if you already have Mirror installed, you are well on your way to file security. Feel free to skip ahead to Quick Start, or open Mirror and explore for yourself. In as little as 3 clicks, your backup will be up and running!

2 ⟡ INSTALLATION

Pre-requisites...

Mirror requires that the following be installed on your machine
  • Windows 7 (or greater)
  • Java SE Runtime Environment 8 (or greater)
Install Windows 7, Windows 8.1, or Windows 10.
Install Java SE Runtime Environment 8 from here.


Installation...
  1. Go to Mirror's homepage and click Download Installer to begin the download.
  2. Double-click the downloaded file Mirror-1.0.0.jar.
  3. Follow the on-screen instructions, saving the program to a location of your choosing.
  4. Double-click the file named Mirror.jar to begin using Mirror!

3 ⟡ QUICK START


 Mirror's one and only window 


Follow these simple step-by-step instructions to create your first backup:
  1. Turn scheduled backup/sync ON or OFF.
  2. If sync is ON, choose your preferred date and time for backups to occur or skip to step 4.
  3. Choose how often you would like to backup by selecting a frequency.
  4. Add source file(s) / folder(s) by clicking the appropriate buttons. To remove an item from your list, select it and click the "Remove Selection" button.
  5. Give your backup a name or leave the default name which will append the current date and time to "Backup".
  6. Click the "Browse..." button to select where your backup file will be saved.
  7. Click the "Save" button to run backups according your schedule or "Run" to backup immediately.

6 ⟡ KNOWN ISSUES

  • Backups are not differential (should only backup new or modified files/folders)
  • Backups are not compressed
  • Backups are not encrypted
  • Scheduled backups are unavailable
  • The source list should accept folders
  • There is no backup log
